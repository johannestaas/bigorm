bigorm
======

A BigQuery ORM


Installation
------------

Install python3.10 via homebrew:

    brew install python@3.10

Ensure `python3.10` is available in your PATH.

From the project root directory:

    make install


Usage
-----

Use --help/-h to view info on the arguments::

    $ bigorm --help
